## üìñ About The Project

This repository hosts the source code for my personal portfolio website. It is designed as a minimalist, dark-themed single-page application that showcases my academic journey, technical projects, and engineering skills.

### Key Features
* **Immersive Design:** Custom CSS-driven "Starfield" background animation.
* **Responsive:** Fully optimized for mobile, tablet, and desktop views.
* **Performance:** Built with **Vanilla JavaScript** and **CSS** (no heavy frameworks) for lightning-fast load times.
* **Interactivity:**
    * Scroll-reveal animations using the `IntersectionObserver` API.
    * One-click "Copy to Clipboard" email feature.
    * Dynamic "Go to Top" navigation.
